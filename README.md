# some-free-useful-site
A suite of some web utilities featuring 100% client-side PDF tools, developer editors, financial calculators, and smart image processing built with Vite.

## 🌐 Product Ecosystem

This repository hosts a suite of 13 static, client-side web utilities optimized for speed and privacy:

### 🔒 PDF & Document Tools (100% Client-Side Privacy)
* **[PDF Redact](https://senaskat.com)** - Secure text redaction.
* **[PDF Edit](https://senaskat.com)** - Zero-upload document editing.
* **[PDF Merge & Split](https://senaskat.com)** - File manipulation utilities.
* **[PDF Extract](https://senaskat.com)** - Table and image extraction.
* **[LockPDF](https://senaskat.com)** - Passwords, unlocking, and watermarks.

### 💻 Developer Tools & Code Editors
* **[ProQuiz](https://senaskat.com)** - Coding language quizzes.
* **[JSON Viewer](https://senaskat.com)** - Formatting and data visualization.
* **[JS Editor](https://senaskat.com)** - Beginner-friendly JavaScript playground.
* **[Python Editor](https://senaskat.com)** - Beginner-friendly Python playground.

### 📈 Financial Calculators
* **[POS Calculator](https://senaskat.com)** - Indian Post Office interest returns.
* **[WPC Calculator](https://senaskat.com)** - SIP and Loan EMI estimations.

### 🛠️ Smart Utilities
* **[Blur Image](https://senaskat.com)** - Manual and AI face blurring.
* **[Make QR Code](https://senaskat.com)** - Text, SMS, and Wi-Fi codes.
